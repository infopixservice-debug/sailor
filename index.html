💡 Barter Tips

🎯 Strategia di base:
• Parti con LV1 goods che costano poco e ne hai accesso via coltivazione/gathering
• Spingi verso T3-T5 tramite rotazioni 1x2 o 1x3 per massimizzare quantità
• T5 Margoria vale 10M silver/barter: prioritario se hai accesso ai NPC
⚠️ Gestione Parley: Parley è il bottleneck reale. Aumenta la Barter Mastery per ridurre il costo per trade, e pianifica le route partendo dagli NPC con costo più basso.
✅ Stock consigliato: mantieni storage bilanciato T1-T5. Il calculator di Serenity suggerisce 83+ fields di storage a Iliya per setup completo, con Ancado (T5) per stoccaggio finale.

💡 Tips & Tricks

🎯 Migliori sailor (tier list community 2025):
• Innocent — Il migliore per speed growth (popolare per sail-and-hold)
• Ambitious — Ottimo per bartering (peso basso + speed alto)
• Born-in-the-Sea — Massima speed base (ma peso alto = meno barter goods)
• Powerful — Balance stats, ottimo con First Mate Proix
• Honest — Cabin cost basso (7), perfetto per riempire slot
⚠️ Attenzione al peso: nave overweight = super lenta + skill disabilitate. Goblin (100-300 LT) sono i migliori per bartering, Giant (500 LT) per pura velocità.
✅ Setup consigliato Carrack post-2025 (110 cabin + slot specializzati):
• Sail slot → il tuo miglior Speed/Accel sailor (es. Innocent lv10 con buona Endurance)
• Wheel slot → miglior Turn/Brake sailor (es. Confident o Strong)
• Cannon slot → Quick-Witted / Quick / Realistic / Curious a seconda del focus
• Deck slot → sailor con cabin cost alto (es. Born-in-the-Sea = 10 cab = +100k durability)
• Mess slot → stesso discorso (cabin alto = più rations)
• Fish slot → qualsiasi sailor sano quando hai l'Oceanbound Otter Rod
• First Mate → Proix per Great Ocean (compass gratis), Cleia per bartering (-10% parley), Tranan per combat (auto-repair)
• Cabin → riempi con Innocent/Ambitious per bartering, o mix per balance
🎮 NUOVO — Usa l'Auto-Assign Feature: dal Wharf Manager → Manage Sailors → Auto-assign. Il gioco piazza automaticamente i tuoi sailor con le stats più alte negli slot giusti. Poi salva come Preset (separato per ogni variante Carrack).
🎯 Barter Method 2025 (Top-Down):
1. Clear tutti i T5 disponibili per primi (priorità assoluta — più Crow Coin + EXP)
2. Poi T4, poi T1-T3 con lo stockpile rimanente
3. Stockpile rules: tieni 2+ di ogni T5 a Ilya, 3+ a Ancado per mai "bloccare" una catena
4. T5m (Margoria, 10M silver) = catena a valore più alto se hai Carrack + Parley sufficiente
5. Refresh solo quando almeno 50% dei barter sono consumati (max value del punto refresh)
